California Map Editor

How to use:
1. Load a California reference image
2. Coastline mode: click to trace the coast
3. Contour mode: click to draw interior detail lines
4. Regions mode: select a location, click to draw its boundary polygon. Each click adds a point; the polygon auto-closes.
5. Drag any handle to adjust — right-click to delete a point
6. Copy output and paste into forest-map.js
